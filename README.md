# 🧠 Deepfake Attack Detection System

## 📌 Overview
This project focuses on detecting deepfake images using deep learning models such as MobileNet and MesoNet. It aims to identify manipulated facial images and classify them as **real or fake** with high accuracy.

## 🎯 Features
- Detects deepfake vs real images
- Uses pretrained CNN architectures (MobileNet, MesoNet)
- Displays confusion matrix and training results
- Supports image-based inference
- Lightweight and efficient model

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## 🧠 Models Used
- MobileNet (custom trained)
- MesoNet (deepfake detection specific architecture)
